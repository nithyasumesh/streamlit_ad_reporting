This example Streamlit app showcases the reports generated by the [dbt_ad_reporting](https://github.com/fivetran/dbt_ad_reporting) data model using simulated data from a fictional office supply company. - Amazon Ads

- LinkedIn Ads
- Microsoft Ads
- Pinterest Ads
- Reddit Ads
- Snapchat Ads
- TikTok Ads

**This is an example report for demonstration and exploratory purposes only.** It demonstrates how easy it is to find insights from the materialized tables produced by the Fivetran Ad Reporting dbt packages.
